# Анкета для роботи над сайтом — Яна Баранова

Інтерактивна анкета (HTML+CSS+JS, без залежностей) для збору стратегічних вступних
даних перед оновленням сайту. Питання розгалужуються залежно від відповідей;
перед надсиланням респондент переглядає й може відредагувати відповіді.

**Жива сторінка:** https://gregory-ivd.github.io/yana-site-survey/

Доставка відповідей — через Google Apps Script (Telegram + e-mail).
URL ендпоінта вписується в `CONFIG.endpoint` усередині `index.html`.
Поки ендпоінт порожній — сторінка пропонує зберегти відповіді у `.txt`.
